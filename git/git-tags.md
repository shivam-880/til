
# Create a tag

```
git tag <tag name>
```

# Delete a tag

```
git tag --delete v.3.0.0
```

# Delete a remote tag

```
git push --delete origin v6.0.2.75
```

# Push tags created locally
```
git push --tags origin master
```

# Push single tag created locally

```
git push origin <tag name>
```

# Fetch all remote tags

```
git fetch --tags
```
